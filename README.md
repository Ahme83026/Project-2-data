# Project 2: Impact of High School Curriculum on GPA

## Description
This project studies whether high school curriculum affects GPA among Computer Science students.

The study compares students from different high school systems, including Thanaweya Amma, British, American, and Other curricula.

## Research Question
Does high school curriculum affect GPA among Computer Science students?

## Files Included
- Dataset file
- Jupyter Notebook file
- Project Report PDF

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Summary
The analysis showed that Thanaweya Amma students had a higher average GPA than British and American curriculum students. The project also includes charts such as a pie chart, bar chart, histogram, and box plot.
